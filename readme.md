# Express User Authentication w/ Passport & Mongoose

A user authentication app that stores users in a local MongoDB server. Session is stored in a cookie so users can close the tab and still be logged in.

## How to run

1. clone this repository
2. Run `npm install`
3. Run `npm start`
4. Go to [localhost:8080](localhost:8080) to register/login

### Dependencies

* Express
* BodyParser
* Cookie-session
* Mongoose
* Passport
* Passport-local
* Passport-local-mongoose
* Pug

### DevDependencies

* Nodemon
